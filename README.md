Convert Animation-GIF App
=======================
## Setup in Mac
```
$ phpbrew install 7.1.0 +default +mysql +fpm +openssl +curl +openssl=/usr -- --with-libdir=lib64
$ phpbrew ext install gd -- --with-jpeg-dir=/usr/local/Cellar/
```
